마이크 접근 권한 필요

실시간 소음 측정을 위해 마이크 사용 권한이 필요합니다.
아래 버튼을 누르면 브라우저의 권한 요청 팝업이 표시됩니다.
팝업에서 허용을 선택해 주세요.

📌 권한 허용 방법
• Chrome/Edge: 주소창 왼쪽 🔒 아이콘 → 마이크 → 허용
• Safari: 환경설정 → 웹사이트 → 마이크 → 허용
• 모바일: 팝업에서 "허용" 탭

마이크 권한이 차단되었습니다

마이크 접근이 거부되어 측정을 시작할 수 없습니다.
아래 안내에 따라 브라우저 설정에서 권한을 허용한 뒤 페이지를 새로고침해 주세요.

🔧 권한 복구 방법
Chrome / Edge:
  주소창 왼쪽 🔒 자물쇠 아이콘 클릭
  → 마이크 → 허용 선택 → 새로고침

모바일 Chrome:
  주소창 오른쪽 ⋮ 메뉴 → 사이트 설정
  → 마이크 → 허용 선택
