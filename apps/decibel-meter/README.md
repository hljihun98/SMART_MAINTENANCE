# 벨트 장력계 v2

마이크로 벨트 진동음을 녹음해 FFT 주파수 분석으로 장력(N)을 계산합니다.  
호기별 세션 관리·PDF 보고서·현장 맞춤 노이즈 저감을 지원하는 2.0 버전입니다.

## 원리

1. 마이크가 벨트를 두드리는 소리를 녹음합니다.
2. Web Audio API AnalyserNode가 실시간 FFT를 수행합니다.
3. 지배 주파수 `f`와 벨트 파라미터(단위 질량, 폭, 스팬)로 장력을 계산합니다:

   ```
   T [N] = 4 × m × w × L² × f² × 1e⁻⁹
   ```

## 주요 기능

- 실시간 FFT 주파수 검출 (10 ~ 5000 Hz), FFT 크기 선택 가능 (8192 / 16384 / 32768)
- 장력 자동 계산 (보정계수 CF 적용) + 목표 장력 게이지 (OK/NG 판정)
- **현장 소음 맞춤 노이즈 저감** — 스펙트럴 서브트랙션 방식
  - 측정 시작 후 충격 없는 구간의 현장 소음을 3초간 자동 학습
  - 학습 완료 후 배경 스펙트럼을 실시간 차감하여 벨트 주파수 SNR 개선
  - 필터 강도 1~4: 노이즈 제거 강도(alpha)와 충격 감지 임계값(triggerK)이 연동
  - 학습 진행 칩(`학습 중 0%` → `학습완료`) 및 재학습 버튼 제공
- **적응형 충격 감지** — 주변 RMS EMA 기반 동적 임계값 (`ambientRms × triggerK`)
- 목표 주파수 잠금 모드 (예상 주파수 ±% 대역만 탐색)
- 측정 플로우 모드: 호기별 세션·벨트 ID(ID1~ID6)·회차 관리 + 세션 진행 표시
- 단순 모드: 직접 저장 + 단순 CSV
- 프리셋: 구동모듈 벨트(170 N ±20% · 2.8 g/m) / 리프팅 벨트(500 N ±10% · 4.2 g/m)
- 측정 기록 CSV / PDF 보고서 내보내기
- 타이밍 가이드 (사인파 주기 3/4/5초 선택) + 파형 시각화
- 모바일 최적화: sticky 상단바, fixed 하단 액션 버튼, 현재 단계만 표시

## 노이즈 저감 상세

### 학습 흐름

```
노이즈 저감 ON (측정 전)
  → 칩: "측정 시작 후 학습"

측정 시작
  → 칩: "학습 중 0%"
  → 벨트 충격 없는 구간에서 현장 배경 스펙트럼 EMA 누적
  → 충격 발생 시 타이머 일시 정지 (여진 오염 방지, 충격 후 2500ms 쿨다운)

3초 누적 완료
  → 칩: "학습완료"
  → 이후 배경 스펙트럼 실시간 차감 + 느린 드리프트 추적 (beta=0.92)
```

### EMA 파라미터

| 상태 | beta | 설명 |
|---|---|---|
| 학습 중 | 0.60 | 빠른 수렴 (신규 데이터 40% 반영) |
| 학습 완료 후 | 0.92 | 느린 드리프트 추적 (신규 8%) |

### filterStrength 연동

| 강도 | alpha (노이즈 차감) | triggerK (충격 임계 배수) | 용도 |
|---|---|---|---|
| 1 약 | 1.0 | 2.5 | 청정 환경 |
| 2 중 | 1.5 | 4.0 | 일반 환경 (기본값) |
| 3 강 | 2.5 | 6.0 | 소음 환경·모바일 |
| 4 최강 | 3.5 | 8.0 | 고소음·모바일 |

### Confidence 계산

노이즈 저감 학습 완료 후, Confidence SNR 계산에 학습된 프로파일을 noise floor로 사용합니다.  
(원본 raw 스펙트럼 사용 시 배경 소음이 포함되어 SNR이 과소평가되는 문제 수정)

## 파라미터

| 상수 | 값 | 설명 |
|---|---|---|
| `MIN_FREQ` | 10 Hz | 탐색 하한 주파수 |
| `MAX_FREQ` | 5000 Hz | 탐색 상한 주파수 |
| `BROADBAND_RATIO` | 0.3 | 광대역 판정 비율 |
| `LEARN_DURATION_MS` | 3000 ms | 노이즈 프로파일 학습 필요 시간 |
| `G` | 9.80665 | 중력 가속도 (m/s²) |

## 데이터 저장

측정 기록은 `localStorage`에 JSON으로 저장됩니다.  
저장 키: `version`, `flowEnabled`, `robots`, `simpleHistory`, `filterStrength`
